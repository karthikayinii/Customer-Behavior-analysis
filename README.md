# Data Analytics Project

## Overview

This project focuses on analyzing a business dataset using Python, SQL, and Power BI to uncover meaningful insights and support data-driven decision-making. The workflow includes data loading, cleaning, exploratory data analysis (EDA), SQL-based analysis, dashboard development, and reporting.

## Dataset

The dataset contains business-related information such as customer demographics, transactions, product details, sales performance, and other relevant attributes. It is used to identify trends, patterns, and key performance indicators (KPIs).

## Tools & Technologies

* **Python** (Pandas, NumPy, Matplotlib, Seaborn)
* **SQL** (PostgreSQL / MySQL / SQL Server)
* **Power BI**
* **Excel**
* **Gamma** (Presentation Creation)

## How to Run

1. Clone the repository:

```bash
git clone <repository-link>
```

2. Install required Python packages:

```bash
pip install pandas numpy matplotlib seaborn sqlalchemy
```

3. Load the dataset into Python.

4. Execute the data cleaning and EDA scripts.

5. Import the cleaned dataset into PostgreSQL/MySQL/SQL Server.

6. Run the SQL scripts provided in the project.

7. Open the Power BI (.pbix) file to explore the dashboard.

8. Review the project report and presentation for detailed insights.
